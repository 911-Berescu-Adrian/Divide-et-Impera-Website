Algoritmi

Mergesort

Este o metodă eficientă de sortare a elementelor, bazată pe interclasarea jumătăților ordonate ale unui vector.

Complexitatea acestei metode este O(n*logn).

Quicksort

Este de asemenea o metodă eficientă de sortare a unui vector.

Se alege un pivot, se ordonează elementele listei, astfel încât toate elementele din stânga pivotului să fie mai mici sau egale cu acesta, iar toate elementele din dreapta să fie mai mari sau egale cu pivotul. Această procedură se repetă la stânga și la dreapta elementului ales.

Complexitatea algoritmului este O(nlogn) și O(n^2) în cel mai nefavorabil caz.

Căutarea binară

Este cel mai eficient algoritm găsire a unui element într-un vector ordonat. cu o complexitate de O(logn).

Se verifică la fiecare pas elementul din mijlocul unei porțiuni de vector, porțiune ce se micșorează la fiecare pas. Dacă elementul căutat este mai mic decat mijlocul, atunci căutăm la stânga, atlfel căutăm la dreapta.

Complexitatea algoritmului este O(logn).
